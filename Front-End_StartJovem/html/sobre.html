Sistema de Gestão de Aprendizes

Uma plataforma desenvolvida para facilitar o gerenciamento de turmas, aprendizes e atividades acadêmicas de forma simples, moderna e eficiente.

Objetivo

Organizar e automatizar o controle de alunos (aprendizes), turmas e vínculos entre eles, permitindo maior eficiência na gestão educacional.

Tecnologias
Node-RED MySQL JavaScript HTML5 CSS3 API REST
Funcionalidades
  • Cadastro de aprendizes e gestores
  • Criação de turmas
  • Vínculo de alunos em turmas
  • Listagem dinâmica via API
  • Dashboard interativo
  • Monitoramento de produtividade
Equipe

Desenvolvido com dedicação para facilitar o aprendizado e gestão de turmas.
